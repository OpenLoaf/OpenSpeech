# 产品定位

## 一句话描述

**按住快捷键说话，松开即把文字写入当前应用的输入框。**

## 目标用户

- 需要大量文字输入的知识工作者：写代码、写邮件、写文档、回复消息
- 中英文混合输入用户
- 对输入法联想 / 拼音打字效率不满意，希望用口述代替打字的用户

## 核心价值

1. **全局可用**：在任何应用的任何可输入区域都能使用，不绑定编辑器或浏览器。
2. **零学习成本**：用户只需要记住一个快捷键。
3. **隐私保守**：录音不持久化、不上传审计；转写结果仅保存在本地设备。
4. **模型自选**：
   - 默认走 OpenLoaf SaaS 云端（按积分扣费，捆绑 OpenLoaf 套餐后无限使用）
   - 也支持 BYO-Model：用户自行配置 REST 端点，不经 SaaS，不消耗积分
   - 详见 `subscription.md`
5. **全平台同步交付**：Windows / macOS / Linux **同版本同步发布**（竞品 TypeLess 仅 macOS + 移动端，无 Linux）。

## 与 TypeLess 的差异化

| 维度 | TypeLess | OpenSpeech |
|---|---|---|
| 平台 | macOS、Windows、iOS、Android（无 Linux） | Windows、macOS、**Linux** 三端同步 |
| 模型 | 内置托管模型，用量计费 | 用户自带 REST 模型端点，费用自担 |
| 计费 | 订阅制（8,000 字/周 Free，$12–$30 Pro） | 软件本身免费 / 开源友好 |
| 透明度 | 设备上限、退款、冲突规则多数未公示 | 所有业务规则在 docs/ 中公开 |

## 非目标（当前版本不做）

- 不做离线模型内置（模型由用户通过 REST 配置）
- 不做移动端（仅桌面 Windows / macOS / Linux）
- 不做多人协作 / 云同步
- 不做音频长期存储
- 不做会议转写、实时字幕等长录音场景

## 竞品参考

- **TypeLess**、**Wispr Flow**、**Superwhisper**
- 差异点：OpenSpeech 允许用户自带大模型（BYO-Model），模型账单由用户自己承担。
