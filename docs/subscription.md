# 订阅与计费规则

> **MVP 阶段定位**：OpenSpeech 本体为 **免费 / 可开源使用**；大模型调用费用由用户直接支付给其选择的 STT 服务商。
> 本文档为商业化迭代时的规则蓝图，MVP 不实现计费系统。

## MVP 阶段

- 软件本体免费，无功能限制、无字数上限、无试用天数。
- 用户需自行申请并支付大模型 REST 服务的费用。
- 不接入支付渠道，不收集订阅信息，**不存在账户体系**（或仅可选用于云端同步词典与设置）。

## 未来商业化（若采用 Freemium 模式）

以下为备选方案，不作为 MVP 承诺。

### 方案 A：订阅托管模型

| 计划 | 价格 | 规则 |
|---|---|---|
| Free | $0 | 每周 8,000 字上限，使用官方内置的标准 STT 模型 |
| Pro（年付） | $10/月 | 无字数上限，支持更高精度模型与自动润色 |
| Pro（月付） | $20/月 | 同上 |
| 试用 | 首次注册 30 天 Pro | 到期自动降级为 Free，不自动扣款 |

### 方案 B：一次性授权（Lifetime License）

| 计划 | 价格 | 规则 |
|---|---|---|
| Lifetime | 一次 $99 | 终身免费使用，包含大版本更新 |
| 仅激活本机，不跨设备 | 允许最多 3 台设备激活 | |

### 方案 C：BYO-Model 永久免费 + 托管增值（**推荐**）

- 软件本体永久免费、功能不阉割，用户配置自己的 REST 端点。
- 官方可选提供"托管 STT 服务"，按用量（字数或分钟）计费，作为增值选项。
- 账户体系仅用于托管服务用户。

## 通用规则（若启用订阅）

1. 订阅随时可在 App 内取消；取消后剩余周期仍可使用。
2. 退款政策：购买后 14 日内无条件退款。
3. 单账号最多激活 **3 台设备**；超出时需在账户页主动解绑。
4. 团队套餐支持最少 3 席起订；管理员可分配与回收席位。
5. 所有字数统计以"转写成功后实际写入历史的字符数"为准；取消与失败不计。
6. 字数统计周期为自然周（周一 00:00 重置，用户本地时区）。

## 礼品卡 / 兑换码

- 兑换码为一次性、实名绑定；一经兑换不可转让。
- 兑换后即刻生效；若用户已在订阅中，延长现有周期。

## 账单渠道

- 支持 Visa / Mastercard / AmEx / Discover。
- 全球定价统一美元；部分地区可按汇率换算显示本地货币，实际扣款仍为美元。
