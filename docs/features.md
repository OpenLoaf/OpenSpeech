# 功能总览

## 核心功能（MVP 必备）

| # | 功能 | 说明 |
|---|---|---|
| F1 | 听写 | 按住快捷键说话，松开写入当前输入框的纯文字转写 |
| F2 | 全局快捷键 | 用户可在设置中自定义组合键；默认 `Ctrl + Shift + Space`（PTT）；详见 [hotkeys.md](./hotkeys.md) |
| F3 | 录音悬浮条 | 录音期间屏幕底部中央显示一个小条（280×56 logical px，TE 风方角），显示功能标签、波形、计时、取消按钮；独立透明 WebviewWindow（`focus:false` 不夺焦）。**MVP 固定光标所在显示器底部中央**；"智能避让焦点输入框 / 用户拖动与按应用记忆位置"列入进阶。详见 [voice-input-flow.md](./voice-input-flow.md) |
| F4 | REST 大模型接入 | 用户在设置中配置 STT 服务端点、API Key、模型名 |
| F5 | 历史记录 | 每次转写结果保存到本地，可查看、复制、删除 |
| F6 | 系统托盘 | 最小化到托盘，托盘菜单含开关录音、打开主窗口、退出 |
| F7 | 开机自启 | 用户可在设置中开启/关闭 |

## 进阶功能（迭代）

| # | 功能 | 说明 |
|---|---|---|
| F8 | 问 AI（Ask AI） | 独立快捷键；语音作为提问，LLM 生成回答写入（TypeLess 不具备，差异化） |
| F9 | 翻译（Translate） | 独立快捷键；语音转写 + 翻译为目标语言后写入（TypeLess 不具备，差异化） |
| F10 | 词典 | 维护专有名词表，提交给大模型作为转写 hints，降低错字率 |
| F11 | 界面多语言 | 至少支持简体中文、English |
| F12 | 音频设备选择 | 用户可指定输入麦克风设备 |
| F13 | 主题 | 跟随系统 / 亮色 / 暗色 |
| F14 | 应用内更新 | 提示新版本并引导升级 |
| F15 | AI 自动润色（可选） | 转写后可选删除 um/uh、口误改口、整理成列表；用户可全局关闭 |
| F16 | 上下文风格 | 根据前台应用调整语气（邮件偏正式、IM 偏口语）；仅作为 prompt 提示传给模型，不改动用户原话含义 |
| F17 | 双触发模式 | Push-to-Talk（按住）与 Toggle（单击开始、再次单击结束）可切换 |
| F18 | 语种自动检测 | 录音的源语种由模型自动识别，用户无需手动切换 |

## 暂不考虑

- 语音合成（TTS）输出
- 实时字幕 / 长录音转写
- 跨设备同步
- 团队共享词典
