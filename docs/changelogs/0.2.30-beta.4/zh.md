## v0.2.30-beta.4

> 测试版，仅 Beta 通道用户可收到。本版本主线是**自定义供应商（BYOK）易用性**：从"配错就直接报错"改为"未配齐自动走云端 + 提示原因"，并引入「已验证 / 未验证」状态，让用户明确知道当前自定义通道能不能用。

### 改进

- **自定义供应商加「已验证 / 未验证」状态徽章**：听写自定义供应商（腾讯 / 阿里）和 AI 改写自定义供应商都新增 `verified` 字段。点「测试」通过 → 标已验证（绿色）；改了 baseUrl / model / 凭证 / vendor / region 等任何会影响实际请求的字段 → 自动重置为未验证（黄色）。卡片右上角直接看到状态。
- **关闭设置时自动回退未验证的自定义通道**：如果你切到了 custom 模式但没把 active provider 测通过就关掉设置面板，会自动写回云端模式并 toast 告诉你"哪一项被回退了"。避免下次按录音键才发现配错了又骚扰一次。
- **未配齐的自定义通道改为静默 fallback 到云端**：以前配了一半（active provider id 缺失 / vendor 缺失 / 列表是空的）按下录音键会直接报 `byok_provider_not_configured` 错。现在改为后端检测到结构性没配好 → 直接走云端转写 + 弹一次 toast 解释「自定义供应商未配齐，已走云端」。区别于"用户主动配过但 keyring 缺凭证"——后者仍然会抛错，避免悄悄掩盖配置问题。
- **AI 改写失败时分流提示**：自定义 AI 供应商失败（缺 ApiKey / 网络错误等）现在弹专属 toast，并提供「切到 SaaS」按钮一键回退；其它错误（如 SaaS 鉴权失败）保持原有提示路径。
- **录音 gate 适配 custom 模式**：以前未登录 OpenLoaf 直接拦住录音流程；现在 custom 模式下 BYOK 自带凭证，跳过登录拦截，能录直接录。离线 gate 仍然适用（腾讯 / 阿里 BYOK 同样走云端 ASR，离线必然失败）。
- **听写默认 segment 模式调整**：设置 → 听写 中 `UTTERANCE`（整段优化）现在排在 `REALTIME`（边说边出字）前面，与上版反过来——多数用户更适合整段优化的输出体验，REALTIME 留给短指令场景。
- **AI 提示词内置示例微调**：默认 refine 提示词去掉显式的 `<processing_inventory>` 列表，改为只通过示例界定边界；新增几条容易踩雷的撤回 / 品牌大小写示例。

### 修复 / 可观测性

- 修复某些情况下 `mode=custom` 但 customProviders 列表为空 / activeId 找不到对应项时，每次按录音键都会触发一次后端 dispatch 失败 + toast 骚扰的问题——配置载入时直接静默回写为 saas。
- **inject 链路加日志 + 释放卡住的修饰键**：enigo 在 Windows 上 SendInput 偶发"静默失败"（高权限窗口 / UWP 沙盒 / 部分 IME / 全屏游戏会忽略 fake unicode key event 但 SendInput 自身返回成功），事后只能看屏幕没字。这版给 `inject_paste` / `inject_type` / 增量注入 / tail 收尾全链路补 INFO/ERROR 日志（含字符数 / 段落数 / 前缀匹配状态 / 16 字预览），并在中途失败时主动 release 已按下的 Cmd/Ctrl/Shift——避免修饰键卡住影响后续输入。日志只打前 16 字预览，不会泄露完整转录。

### 已知风险 / 想让 Beta 用户帮忙验

- 配过腾讯 / 阿里自定义供应商的用户，请重启后进设置面板看一眼徽章状态；如果显示「未验证」，点一次「测试」即可。verified 字段是新增的，老数据默认会被标为未验证。
- 关闭设置面板时如果触发了自动回退 toast，请确认看到的提示语正确（中文 / 英文 / 繁体三种 locale 各自有翻译）。
- 自定义 AI 改写失败的 toast 上的「切到 SaaS」按钮请试着点一下，确认能直接生效不需要重启。
