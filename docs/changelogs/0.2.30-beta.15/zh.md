## v0.2.30-beta.15

> 测试版，仅 Beta 通道用户可收到。本轮重点是**快捷键系统大重构**——左右修饰键区分、新增多种冲突检测、系统快捷键冲突预警；同时把 AI 提示词编辑收进独立 dialog，让设置页清爽。

### 新增

- **快捷键 · 区分左右修饰键**：录入快捷键时按你实际按下的物理键写入「左 / 右」（左 Ctrl ≠ 右 Ctrl）；预览和触发都按物理键判定，避免误触。fn 不分左右。旧绑定 migrate 时统一记为「左」。
- **快捷键 · 新增冲突类型识别**：
  - **Modifier-only 互为子集** → 「会出现幻影录音」（例如「Ctrl」与「Ctrl + Shift」会互相误触）
  - **Modifier-only 与 Combo 子集** → 「按下时会双触发」
  - **双击与共用修饰键** → 「双击窗口会冲突」
- **快捷键 · 系统冲突预警**：组合可能与系统常用快捷键冲突（如 Cmd+Q / Ctrl+C 等）会出黄字提示；F1-F12 单独绑定时提示「在多数 Mac 默认是亮度 / 音量等系统功能键」。
- **历史详情 · 复制调试提示词**：详情页操作菜单新增「复制调试提示词」，把当前条目的原始 ASR + refine prompt 一键复制走，方便外部对照测试。
- **Overlay · BYOK 失败一键回退**：BYOK 转写失败的 toast 多了一颗「切到云端」按钮——点一下就把 provider 切回 OpenLoaf SaaS，再按快捷键重新录就能用。

### 改进

- **快捷键 · Modifier-only 至少 2 个修饰键**：单个修饰键作为快捷键会吞掉这颗键的所有原生组合（按 Ctrl 就触发录音，那 Ctrl+C 也会先吞），现在直接在录入校验阶段拦截，提示「至少需要两个修饰键」。
- **快捷键 · 特殊键白名单调整**：
  - 移除 Tab（系统层面占用太多 / 误触代价过高）
  - 新增 Esc / Enter / 方向键（高级模式下可用）
- **快捷键 · macOS 双击 Fn 拦截**：macOS 双击 Fn 是系统听写默认快捷键，会被系统先吞，现在录入阶段直接拦截。
- **快捷键 · Combo 主键限制**：Tab / Backspace / Delete / CapsLock 不再允许作为主键，误触代价过高。
- **快捷键 · Fn + 主键不可组合**：Fn 键和主键的组合在系统层无法注册，现在录入校验直接拦截。
- **设置页 · 提示词编辑独立 dialog**：「听写优化」「翻译」「会议纪要」三类系统提示词从内嵌 textarea 改成独立 dialog，主设置页只显示「已自定义 / 默认」状态徽章和「修改」按钮。设置页大幅瘦身。
- **设置页 · 润色场景独立 dialog**：润色场景列表也收进 dialog，主页面只显示一行 hint 摘要。
- **快捷键 · 键帽可视化**：HotkeyBinder / HotkeyPreview 用统一的键帽样式渲染，左右修饰键带角标显示。

### 修复

- 录入快捷键时若按下的不是合法组合，现在会出更具体的错误提示而不是笼统的「非法组合」（参考新的 `validation.modifier_only_min_two` / `combo_disallowed_main` / `combo_no_fn` / `double_tap_fn_macos` 等条目）。

### 升级提示

- 本版本是 SemVer 预发布段（`0.2.30-beta.15`），仅 Beta 通道下发；Stable 通道用户不会收到。
- 切到 Beta 通道：设置 → 关于 → 更新通道 → 选 Beta，重启或托盘「检查更新」。
- **快捷键存储 schema v3**：旧绑定升级时左右修饰键统一填「左」，对绝大多数用户行为无感知；如果你的物理键习惯是右 Shift / 右 Cmd，重启后请到设置 → 快捷键里重录一次以恢复物理键差异。

### 想让 Beta 用户帮忙验

- **快捷键左右区分**：设置 → 快捷键，把 PTT 重录一遍——故意用「右 Ctrl」录入，再按「左 Ctrl」尝试录音，预期不触发；按「右 Ctrl」才触发。fn 仍然不分左右。
- **子集冲突拦截**：试着把一个绑定设成「Ctrl + Shift」，再把另一个设成「Ctrl」（modifier-only），预期：第二个绑定弹出「与『xxx』修饰键集合互为子集，会出现幻影录音」。
- **macOS 双击 Fn 拦截**：在设置 → 快捷键里尝试录入「双击 Fn」，预期：弹出「macOS 双击 Fn 是系统听写默认快捷键，会被系统先吞」拒绝录入。
- **历史复制调试提示词**：随便点一条历史 → 详情 → 操作菜单 → 「复制调试提示词」，预期：toast 提示「调试提示词已复制」，粘贴后能看到 ASR 原文 + 当时使用的 refine prompt。
- **BYOK 一键回退**：把 ASR provider 切到 BYOK 但故意填错 key → 录一段 → 应当看到失败 toast 上有「切到云端」按钮，点一下后再录一段，确认走的是 OpenLoaf SaaS。
- **提示词独立 dialog**：设置 → AI → 系统提示词，点「修改」打开 dialog，编辑保存后回主页面，确认状态徽章变成「已自定义」；点「重置默认」确认回到「默认」。
