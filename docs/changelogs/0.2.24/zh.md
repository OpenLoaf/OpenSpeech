## v0.2.24

### 新增
- 听写模式新增「AI 优化（推荐）」档并设为默认：在整句听写基础上调用大模型整理 —— 去口头禅、消重、识别自我修正、按顺序词列表化（同语言不翻译）
- 历史详情页支持「AI 优化 / 原始文本」一键切换查看
- 主窗口启动时自动适配屏幕尺寸，小屏 / 高 DPI 不再溢出
- 关于页新增「打开日志目录」按钮，方便排查问题
- Windows 安装包支持简体中文、繁体中文、英文三语界面
- 设置 → 行为新增「历史保留时长」选项（永远 / 90 天 / 30 天 / 7 天 / 不保存）
- 启动检测到新版本时支持「跳过此版本」

### 改进
- 听写模式重命名：原「自动分句」→「实时转换」，原「手动分句」→「整句听写」（语义不变，老用户的选择会平滑迁移）
- 录音文件改用 OGG Vorbis 编码：体积约为旧 WAV 的 1/10；旧版录下的 .wav 仍可读取、导出、重转写
- 听写过程在「思考中…」之后新增「正在输出中…」状态，反映 AI 优化与文本注入进度
- 移除原生窗口装饰，改用自定义标题栏控件（Windows / Linux）；macOS 仍使用系统红绿灯
- 悬浮录音条定位改用屏幕可用区域（排除任务栏），多屏环境下按光标所在屏幕定位
- 快捷键预览图标改为矢量渲染，显示更清晰
- 统一滚动条样式为纤细工业风
- 录音或语音识别启动失败时立即中断整个会话并提示，不再继续无用录音
- Dialog 关闭动画结束后再卸载内容，消除闪烁与合成层堆积
- 悬浮条切换状态时不再闪屏（CSS transition 替代 keyframe）

### 修复
- 修复尾部音频偶尔被截断导致服务端返回 0 段 Final / 积分扣 0 的「假静音」问题
- 0 段 Final 不再静默关闭浮窗 —— 提示「未识别到内容，可在历史里点重试」
- 修复音频线程意外退出后再次录音报「audio stream not running」的问题
- 主窗切换语言后悬浮条偶尔不跟随的兜底处理
- 一系列 dialog / overlay / log 细节修复

### 移除
- 移除尚未上线的「问 AI」「翻译」快捷键入口
- 移除未使用的设置项：翻译目标、语言变体、悬浮条常驻显示
