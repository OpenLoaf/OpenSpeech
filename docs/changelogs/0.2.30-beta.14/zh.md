## v0.2.30-beta.14

> 测试版，仅 Beta 通道用户可收到。本轮是 beta.13 之后累积的一次大放出：**会议模式真实波形 + AI 纪要 + 导出 Markdown + 断网重连**、**统计弹窗维度大改**、**录音抓前台应用名为后续应用统计打底**，外加 AccountDialog 体验修和 License 协议变更。

### 新增

- **会议模式 · AI 纪要**：会议结束/进行中可以一键让 AI 生成 Markdown 格式的纪要（决策、行动项、关键讨论点等），落盘到 `recordings/<日期>/<id>.summary.md`；review 页用 `MeetingSummaryPanel` 渲染，未生成 / 流式生成中 / 已生成 / 失败四态切换。系统提示词支持中文（zh-CN / zh-TW）+ 英文三语，可在「设置 → AI → 系统提示词」第四个 tab "会议纪要"自定义。
- **会议模式 · 导出 Markdown**：会议详情页右上"导出"，弹出系统保存对话框，写入带元数据头（标题 / 时间 / 时长 / 参会人）+ 按发言人分段的 Markdown 文件。
- **会议模式 · 真实波形**：抽出 `AudioWavePlayer` 共享给历史详情和会议 review，支持点击 segment 跳转到对应时间；live 录制中用 `LiveWaveform` 实时画波形（订阅 20Hz audio-level），暂停时冻结最后一帧而不是清零。
- **会议模式 · 断网自动重连**：实时会议遇到网络抖动会自动指数退避重连（0.5s → 1s → 2s → 4s → 8s × 5 次），重连后时间轴和句子 ID 接续不会错乱；重连期间顶部出现琥珀色提示横幅。
- **统计弹窗 · 6 类新分面**：每个 KPI 都换上更相关的 facet 组合——
  - 时长 → **录音时长分桶**（< 10s / 10–30s / 30–60s / 1–2min / > 2min）+ 活跃时段热力图
  - 字数 → session 长度分布 + **类型混合**（听写 / 提问 / 翻译）
  - WPM → 模式 vs 平均 + **Provider 分布**（Tencent / Aliyun / OpenAI / SaaS …）
  - 节省时间 → 等价时长 + **AI 改写覆盖率**（多少 session 走了 refine + 平均 refine 耗时）
  - Sessions → **状态混合**（成功 / 失败 / 取消）+ **ASR 通道分布**（realtime / saas-file / aliyun-file …）

### 改进

- **License 协议变更（MIT → PolyForm Noncommercial 1.0.0）**：源码继续公开，**禁止任何商业用途**（销售、SaaS 转售、面向客户的内部商业系统等）。**个人 / 学习 / 研究 / 评估 / 非营利使用不受影响**。商用授权可以单独联系作者。`README.md`（含英文 / 繁中）与 UI 关于页对应文案已同步。
- **录音元数据 · 抓取前台应用名**：每次录音开始的瞬间记录当时的前台应用（如 Chrome / VSCode / 微信），写入 history。这一版还没在 UI 上展示这一维度，**是为下一版"按应用看用量统计"做铺垫**——任何抓取失败兜底为空，不影响录音。
- **AccountDialog · 长邮箱换行 + 微信昵称 + 高度自适应**：长 email 不再撑破对话框；微信登录后昵称正常显示；不同账号信息长度的对话框高度跟着内容走，不再固定高度。
- **统计弹窗 · 主图布局**：每日柱图 / WPM 折线高度从 `h-44` 缩到 `h-32` 给下方 facet 腾位置；Range 切换器单独一行右对齐，更容易看清楚当前选的是 Today / 7d / 30d / 90d / All。

### 修复

- 不直接修 bug，但 License 切换 + AccountDialog 修都属于上一轮反馈跟进。

### 升级提示

- 本版本是 SemVer 预发布段（`0.2.30-beta.14`），仅 Beta 通道下发；Stable 通道用户不会收到。
- 切到 Beta 通道：设置 → 关于 → 更新通道 → 选 Beta，重启或托盘「检查更新」。
- 数据库迁移：本版包含 `v9 history_add_summary_path`（给 history 表加 `summary_path` 列），首次启动会自动执行；老纪要不会丢，只会随会议详情页生成补齐。

### 想让 Beta 用户帮忙验

- **会议 AI 纪要**：找一段不超过 30 分钟的会议录音，到 review 页点「生成纪要」，预期：流式输出 Markdown，最终落盘并能在重启 App 后再次打开。换成空网（断 WiFi）应当切到失败态，不会卡死。
- **会议导出 Markdown**：在会议详情页右上点「导出」→ 选保存路径 → 用任意 Markdown 编辑器打开，确认元数据头 + 发言人分段都对。
- **断网重连**：会议进行中拔掉 WiFi 5–10 秒再插回，预期：顶部琥珀色横幅出现 → 自动重连成功 → 后续句子能续上、时间轴不跳。
- **统计弹窗新分面**：打开 Stats 弹窗，把每个 KPI（duration / words / wpm / saved / sessions）都点一遍，确认右侧 facet 内容跟着切换、Donut 图例数值齐全。
- **License 文案**：设置 → 关于，看「许可证」是否显示 PolyForm Noncommercial 1.0.0；README 顶部 / 底部和 promo 页 footer 也对应同步。
- **AccountDialog**：装一个超长邮箱账号 / 用微信登录，确认对话框不会被撑破或文字遮挡；切换不同登录方式时高度自适应。
