## v0.2.19

### 修复

- 自动更新真的能用了。0.2.18 之前从「检查更新 → 立即安装」走完后，下载完替换了应用包但**老进程没退出**，下次手动重启才能用上新版本——很多用户反馈「明明点了升级，怎么还是旧版」。现在替换完成后会立刻自动重启到新版本，肉眼能看到 OpenSpeech 一闪重启然后版本号变了。
- 自动更新下载卡死兜底。极端网络环境下（GitHub 资源 CDN 重定向慢、TLS 握手挂死等）下载会无声卡住，前端没进度也没失败提示。现在 5 分钟没完成会强制超时，弹失败 toast 让用户知道，而不是干瞪眼。

### 改进（仅影响发版基础设施，终端用户不直接感知）

- 修复 Linux / Windows 用户的 OTA 通道。之前 `latest.json` 只列了 macOS 两个平台 key，Linux / Windows 用户只能从 GitHub Releases 手动下安装包。本次发版起，全部 6 个平台都进入 OTA 列表。
