## v0.2.30-beta.10

> 测试版，仅 Beta 通道用户可收到。本轮主要是**听感与快捷键体验**的优化，附带会议页错误引导与用量统计入口。

### 新增

- **录音提示音**：开始 / 停止 / 取消三种动作都会播一个低音量的短提示音（start / stop / cancel.wav），即使没看屏幕也能确认按到了。可在「设置 → 录音」里关闭。
- **快捷键冲突弹窗**：尝试绑定一个已经被其他动作占用的快捷键时，会弹一个清晰的对话框告诉你"被谁占了"，以及是否一键覆盖；不再静默失败。
- **会议错误码引导**：会议页接入腾讯实时说话人分离时，遇到具体错误码（鉴权失败 / 配额不足等）会弹出引导对话框告诉下一步去哪改。
- **用量统计弹窗**：在设置 → 关于里新增「查看本月用量」入口，可以看到本月听写时长 / AI 改写次数 / 会议时长等统计。

### 改进

- **快捷键绑定面板**重写：原本的 `HotkeyField` 改成新的 `HotkeyBinder`，捕获键位更准、视觉更清晰，触屏 / 修饰键 only 的场景也更稳。
- **悬浮窗（Overlay）状态机**重排：录音 / 转写 / 改写各阶段切换更连贯，少了几处状态错位导致的"卡在 loading"。
- **会议页**：把腾讯说话人分离结果落库 + 接进最近会议列表，可以从首页直接打开过去的会议复盘。
- **i18n 同步**：本轮新文案在 zh-CN / zh-TW / en 三套语言里全部对齐。

### 修复

- **快捷键 modifier-only 绑定**在某些机型释放时偶尔丢事件的小毛病。
- 会议页在网络抖动后偶发的"假阳性 VAD 截断"。

### 升级提示

- 本版本是 SemVer 预发布段（`0.2.30-beta.10`），仅 Beta 通道下发；Stable 通道用户不会收到。
- 切到 Beta 通道：设置 → 关于 → 更新通道 → 选 Beta，重启或托盘「检查更新」。

### 想让 Beta 用户帮忙验

- **提示音**：连续按几次听写快捷键，确认 start / stop / cancel 三种音都能听到，且不刺耳。如果声卡切到外置音箱后异常请反馈。
- **快捷键冲突**：故意把听写和翻译都绑成同一个组合键，看冲突弹窗是否清楚地说明"被谁占了"。
- **会议页**：开一个 1-2 分钟的会议测试说话人分离是否落库 + 能否在首页"最近会议"看到。
